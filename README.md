# Personal Web Page - Jianjun Chen
 
A modern, dynamic, and aesthetic personal landing page built using pure HTML, CSS, and JavaScript. 

## 🌟 Features

- **Real-Time Digital Clock**: Fetches your local system time and updates every single second.
- **Contextual Greetings**: Automatically detects the time of day and greets you with "Good Morning", "Good Afternoon", "Good Evening", or "Good Night" based on your local hour.
- **Glassmorphism Design**: Uses blurred, frosted-glass effects (`backdrop-filter`) to create a stunning, premium visual experience.
- **Ambient Animations**: Drifting, slow-moving CSS gradient orbs in the background create a soothing, motion-rich atmosphere.
- **Interactive UI**: Soft hover effects that smoothly elevate the card to make the website feel responsive and alive.

## 🚀 How to Use

1. Check out this repository or download `jianjun_chen.html`.
2. Simply double-click on `jianjun_chen.html` to open it in any modern web browser (Google Chrome, Microsoft Edge, Firefox, Safari).
3. No web server, Node.js, or complex installation is required to run the core logic (an internet connection is only needed once to load the custom Google Font).

## 🛠️ Built With

- **HTML5:** Clean, semantic document structure.
- **CSS3:** Advanced styling including CSS Variables, Flexbox, Keyframe Animations, gradients, and blur filters.
- **Vanilla JavaScript:** Lightweight DOM manipulation for the dynamic real-time clock and conditional greetings.
